# Ubly Fleet Management 🚗  

## 📌 Información del Proyecto  
Ubly es una empresa que **renta autos** para conductores de **Uber y Lyft** en Illinois. Esta aplicación web permite gestionar la flota de vehículos, incluyendo su disponibilidad, estado y asignación a conductores.  

## 🎯 Objetivos  
- Mostrar información sobre cada vehículo en la flota.  
- Facilitar la administración de disponibilidad y estado de los autos.  
- Optimizar la gestión de conductores asignados a cada vehículo.  

## 🏗️ Estructura del Proyecto  
📂 `ubly-fleet/`  
├── 📁 `pages/` → Contiene las páginas principales.  
│   ├── 📝 `index.js` → Página de inicio.  
│   ├── 📝 `fleet.js` → Página para la gestión de flotas.  
├── 📁 `components/` → Componentes reutilizables.  
│   ├── 📝 `FleetCard.js` → Componente de vehículo individual.  
│   ├── 📝 `FleetList.js` → Lista de vehículos.

├── 📄 `.gitignore` → Lista de archivos ignorados por Git.

├── 📄 `README.md` → Documentación detallada.
